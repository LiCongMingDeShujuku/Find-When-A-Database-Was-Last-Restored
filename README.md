## 中文
这个简单的脚本将显示数据库还原的时间。 可以帮助你找到任何或所有数据库的还原历史记录。


## English
This simple script will show you when databases were restored. It helps you find the restore history of any or all databases.

---
## Logic
```SQL
select
user_name
, convert(char, restore_date, 9) as 'when restored'
, restore_date
, destination_database_name
, restore_type
, replace
, recovery
from
msdb..restorehistory
order by
restore_date desc


```
